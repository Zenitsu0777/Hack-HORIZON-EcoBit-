## ✨ Features

- 📊 **Interactive Visualizations** — Real-time charts and graphs using Chart.js and Recharts.
- 🧠 **Smart Forms** — Dynamic and validated inputs with React Hook Form + Zod.
- 🔐 **User Authentication** — Secure login/registration via Passport.js.
- ⚙️ **Backend Powered** — Robust API built with Express and Drizzle ORM.
- 📱 **Responsive Design** — TailwindCSS ensures mobile-first, consistent UI.
- 📆 **Energy Usage Insights** — Analyze your energy consumption patterns by time, category, and cost.
- 🌐 **Seamless Navigation** — Lightweight routing via Wouter.
- 📦 **Optimized Dev Experience** — Vite, TypeScript, and React Query for lightning-fast DX.
